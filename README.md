## Deployment 🚀

Start web server

```bash
yarn && yarn build && yarn start
```

1. Install deps

```bash
yarn
```

2. Build app

```bash
yarn build
```

3.  Run local server at [`localhost:3000`](localhost:3000)

```bash
yarn dev
```
